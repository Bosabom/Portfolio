Hotel management system

The web application "Hotel" allows the user to manage the hotel (opening, closing, resuming work), open new rooms for the possibility of check-in. You can monitor the situation in the hotel: which rooms are occupied/free, and if occupied, for how many days. You can also make a report for a specific day of the hotel's work. Tech-stack: C#, ASP.NET MVC, EF Core, LINQ, Web API, Razor Pages, Bootstrap, AutoMapper, Moq, Postman.

This is my version of 15-puzzle-game. The game "Fifteen" is a classic puzzle. It is a set of identical tiles with numbers from 1 to 15, and there is also one empty cell. All this forms a 4x4 field.
The goal of the game is to move the tiles, to achieve their order by numbers, preferably making as few moves as possible. Tech-stack: HTML, CSS, JavaScript.
