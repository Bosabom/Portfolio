Портфолио

Project "Hotel"

Оконное приложение «Hotel» позволяет пользователю управлять работой отеля (открытие, закрытие, возобновление работы), открывать новые комнаты для возможности заселения. Можно наблюдать за ситуацией в отеле: какие комнаты заняты/свободны, и если заняты, то на сколько дней. Так же можно сделать отчет за определенный день работы отеля.
Проект реализован на языке C# и при помощи технологии Windows Forms.
Repository

Task Manager

«Task manager» - оконное приложение, которое дает возможность пользователю просматривать список активных процессов компьютера, а так же информацию про каждый процесс. В добавок, пользователь может выбрать процессы из этого списка, за которыми надо наблюдать.
В процессе наблюдения на экране будут появляться оповещения об открытии или закрытии процесса из списка выбранных.
Проект реализован на языке C# и при помощи технологии Windows Forms.

15 puzzle-game

Игра «Пятнашки» - классическая головоломка. Представляет собой набор одинаковых костяшек с нанесенными числами от 1 до 15, а также есть одна пустая ячейка. Все это образует поле 4x4.
Цель игры — перемещая костяшки, добиться упорядочивания их по номерам, желательно сделав как можно меньше перемещений.
Проект реализован с применением технологий HTML, CSS, JS.
